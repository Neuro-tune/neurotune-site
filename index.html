All Commands

/play

Play a track or playlist.

/pause

Pause the current track.

/resume

Resume paused music.

/skip

Skip to next track.

/queue

Show current queue.

/nowplaying

Display track that’s playing.

/mood

Generate playlist by mood.

/filter <name>

Apply audio filter.

/lyrics

Show lyrics for current track.

/profile

View your music profile.

/stats

Show listening stats.

/top

Show top tracks/artists.

/ping

Check bot latency.

/support

Get support server link.

/invite

Get bot’s invite link.

/help

Show help menu.

Which platforms are supported? — Supports Spotify, YouTube, SoundCloud, Deezer, Apple Music, and more.

How can I apply filters? — Use /filter <name> with names like bassboost, nightcore, 8D.
